# Exame App

Landing page com captação de leads para o Exame App, um aplicativo que centraliza a vida médica do usuário em um único lugar, permitindo armazenar exames, receitas e vacinas de forma organizada.

---

## O que foi feito

- Landing page responsiva com as secoes: hero, problema, solucao, beneficios e formulario de cadastro
- Formulario de captacao de leads com validacao de campos no frontend e no backend
- Painel administrativo com autenticacao, listagem de leads, busca, ordenacao e paginacao
- API REST com endpoints para criacao de leads e gestao do painel admin
- Banco de dados com persistencia dos leads cadastrados
- Deploy da aplicacao em producao com links publicos

---

## Linguagens e tecnologias

- TypeScript
- JavaScript
- HTML
- CSS
- SQL

## Frameworks e bibliotecas

- Next.js (frontend)
- Node.js com Express (backend)
- Tailwind CSS (estilizacao)
- PostgreSQL com node-postgres (banco de dados)
- JSON Web Token (autenticacao do painel admin)

---

## Plataformas de deploy

- Vercel: hospedagem do frontend
- Render: hospedagem do backend
- Neon: banco de dados PostgreSQL em nuvem
